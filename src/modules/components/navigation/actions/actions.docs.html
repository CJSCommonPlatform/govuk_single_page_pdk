Action bars

Toggled details of contested issues.

 Details of action.
The pdk-action components are used for displaying the state of a workflow that has one or more possible outcomes.

pdk-action-start

  • label The text to be displayed on the action bar.
  • on-start (Optional) An expression to be invoked when the component is clicked.
  • disabled (Optional) An expression to determine whether the on-start functionality be disabled.
  • invert (Default: false) A boolean value for using the component's darker design.

pdk-action-complete

  • label The text to be displayed on the completed action bar.
  • edit-label (Default: Amend) The text to be displayed on edit link..
  • on-edit (Optional) An expression to be invoked when the component is clicked. Note that this will override any default action defined on the pdk-action.
  • highlight (Default: false) A boolean value for setting whether the status flag be displayed for the completed action. Note that this will override any default defined on the pdk-action.
  • locked (Default: false) A boolean value for setting whether the action can be amended.
  • highlight-label (Default: Saved) The text to be displayed when the highlight flag is true.
  • disabled (Optional) An expression to determine whether the on-edit functionality be disabled.

pdk-action-details

  • label The text to be displayed on the summary label.
  • open (Default: false) An expression to expand the details of the action.
