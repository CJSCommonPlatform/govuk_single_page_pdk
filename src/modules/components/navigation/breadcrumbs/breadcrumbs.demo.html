Breadcrumbs

Each element of the breadcrumb has a text attribute that defines the text of links and the final element (current page)

Each of the link elements has a state attribute that takes the state for the link as defined in the $stateProvider routing. In other words it is the same as you would pass to ui-sref.

  1. Platform Development Kit
  2. Components
  3. Navigation
 breadcrumbsData = [ { state: 'home', text: 'Platform Development Kit', }, { state: 'components', text: 'Components' }, { text: 'Navigation' } ];
