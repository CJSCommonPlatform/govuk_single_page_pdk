Right column navigation

Component for a column navigation

Each component has an optional metadata attribute. Metadata is displayed at the top of the component. Metadata can either be passed in as an object (for a single metadata item), or as an array. Metadata can render a string of html (see example).

Each component has a required sections attribute which takes an array of sections.

Each section has an optional header string and a required list of links. Links can be made bold by adding an isBold: true property to the particular link object.

  • View this user’s details
  • Search for user in this organisation
  • Search for another organisation
  • More
