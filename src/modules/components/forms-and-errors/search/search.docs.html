Search

The pdk-search component is a container for inputs that perform searches.

Configuration

  • on-search An angular expression to be invoked when the user presses the enter key or clicks the search button. The search text is exposed to the expression via the $event variable.
  • invert A boolean expression for using the component's inverted colour scheme. This is typically used on non-white backgrounds.
  • inline (Default: true) A boolean expression for displaying the search as an inline or block element.
