Date Input

The pdk-date-input component is a form control used for entering dates in a dd/mm/yyyy format.

Configuration

  • date-input-format A string format with which to parse the date prior to it being set as the model value. This uses the date notation according to the Angular Date filter.

Core Validators

  • The dateFormat error exists when the entered date is incomplete or does not adhere to the expected format.
  • The dateExists error exists when a date of the valid format is not recognised as valid.

Optional Validators

  • date-input-max A maximum date against which to validate the entered date. This may be a date object or string representation of a date.
  • date-input-min A minimum date against which to validate the entered date. This may be a date object or string representation of a date.
  • date-input-future A boolean expression as to whether the entered date must be in the future.
  • date-input-past A boolean expression as to whether the entered date must be in the past.
