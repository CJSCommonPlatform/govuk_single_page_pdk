Date Input

A form input for date entry, which binds a valid date to an ng-model using a Date object. This component accepts the following additional attributes:

date-input-format A string format with which to parse the date prior to it being set as the model value. This uses the date notation according to the Angular Date filter.

date-input-max A maximum date against which to validate the entered date. This may be a date object or string representation of a date (for example, 2015-10-10).

date-input-min A minimum date against which to validate the entered date. This may be a date object or string representation of a date (for example, 2015-10-10).
