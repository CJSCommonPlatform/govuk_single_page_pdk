Error Message

The pdk-error-message component works in conjunction with ngMessages to style errors and provide default messages for a range of ngModel validators.

These default messages can be overriden simply by defining a new ng-message for that error key.

Note, the angular-messages library must be loaded to use this component.

Configuration

  • multiple-choice A boolean value as to whether the message belongs to a multiple choice input (like a checkbox or radio group).
