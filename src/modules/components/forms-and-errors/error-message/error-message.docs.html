Error Message

The pdk-error-message component works in conjunction with ngMessages to style errors and provide default messages for a range of ngModel validators.

These default messages can be overriden simply by defining a new ng-message for an error key.

Note, the angular-messages library must be included to use this component.

Configuration

  • type (Default: text)

    The type of input belonging to the error, such that messages will be specific to that type. Valid options are text, radio and checkbox-group.
