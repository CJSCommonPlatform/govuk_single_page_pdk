Alerts

The pdk-alert component is used for displaying messages of high importance.

Configuration

  • banner (Default: false) A boolean expression for displaying the alert so that it spans the width of the viewport.
  • banner-class (Optional) A container class for the banner content. This should be specified when a page's content does not use the conventional GDS width.
  • show-icon (Default: true) A boolean value for setting for whether the accompanying icon be displayed.
