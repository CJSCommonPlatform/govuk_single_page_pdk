Tabs

pdk-tabset

  • vertical (Default: false) A boolean value as to whether the tabs be displayed vertically instead of their horizontal default.
  • on-tab-selected An angular expression to be invoked when the focused tab changes. The tab change information is exposed to the expression via the $event variable.

pdk-tab

  • heading The text to be displayed as the navigation label for the tab.
  • selected (Optional) A boolean value for setting the tab as selected. This can be used to set the default starting tab, or manually set it at a future point. When not provided, the first tab will be selected.

pdk-tab-heading

  • As an alternative to the heading text on a tab, this component can instead be used within a pdk-tab to define HTML content for the heading.
