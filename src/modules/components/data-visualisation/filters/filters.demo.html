Filters

Displays a table of data and buttons with preset filters to filter the table data.

The string, object or function used to defined each filter uses the standard Angular filter expression format described in the Angular filter docs

Each row of table data is an object where the keys match the column name of the table header name property

Links can be applied to any table cell. The onclick handler is described separately from the rest of the row data in the onClicks object. The keys of the onClicks object relate to the column names for that row of data.

  
    // Object for one table row
    {
      name: 'Pending users',
      email: 'john.smith@gmail.com',
      status: 'pending',
      role: 'user',
      onClicks: {
        'name': function() {
          // onClick handler for the name cell of this row
          // Presence of onClick handler will turn the cell data into a link
        },
        'email': function() {
          // onClick handler for the email cell of this row
          // Presence of onClick handler will turn the cell data into a link
        }
      }
    }
