Collapsible Pane

The pdk-collapsible component is used for toggling content.

Configuration

  • collapsed (Default: false) A boolean value for setting the initial collapsed state of the panel.
  • collapse-label (Default: Show) The button text to be displayed when the panel is collapsed.
  • open-label (Default: Hide) The button text to be displayed when the panel is open.
  • collapse An attribute to be applied to one or more child elements such that they will disappear when the panel is in its collapsed state.
