Document Download

Component for displaying a document download

An optional imageSrc attribute will take a url and show the image as a thumbnail instead of an icon.

An optional smallIcon boolean attribute can be passed to display a smaller icon for smaller screens.

The title text will be derived from the filename, unless an options title string attribute is passed which will override the filename
