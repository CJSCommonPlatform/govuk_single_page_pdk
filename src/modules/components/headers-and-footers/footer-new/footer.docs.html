Footer

The pdk-footer component is used as the sole page footer within a page.

Configuration

  • container-class (Optional) A container class for when an alternative content width to the GDS default is required.

Inner components

  • footer-content

    The content to serve as the primary content of the footer.

  • footer-nav / footer-nav-item

    The navigation menu of the footer.
